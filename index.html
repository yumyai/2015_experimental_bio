cat, grep and stuff

grep ">" 01.fa
wc -l 01.fa

Use them together


# same as grep ">" 0.1fa
cat 01.fa | grep ">"
# same as wc -l 0.1fa
cat 01.fa | wc -l

Use them together


grep ">" 01.fa > temp
wc -l temp

More concise


cat 01.fa
cat 01.fa | grep ">"
cat 01.fa | grep ">" | wc -l  # Success
